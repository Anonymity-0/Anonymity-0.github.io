2.变量及标识符

变量及标识符

关键字与保留字

关键字

定义：被JAVA赋予了特殊含义，专门用作字符串的文字

特点：字母全都是小写

保留字

定义：现有JAVA版本尚未使用，但以后可能会作为关键字，避免命名时使用

如：goto，const

标识符

定义：Java对各种变量，方法和类等要素命名使用的字符序列成为标识符

简单来说，凡是自己起名字的地方叫标识符。

规则：

1.由字母，数字，_或$组成

2.数字不能开头

3.不能只是关键字和保留字，但是能包含关键字和保留字

4.严格区分大小写，长度无限制。

5.不能包含空格

命名规范

包名：多单词组成时所有字母都小写

类名、接口名：多单词组成时，所有单词首字母大写 如：

变量名、方法名：第一个单词首字母小写，第二个单词开始首字母大写

常量名：所有字母大写，多单词时用下划线连接

变量

概念：内存中的一个存储区域，该区域可以在同一类型范围内不断变化，该变量是程序中最基本的存储单元，包括变量类型，变量名，和存储的值。

作用：在内存中保存数据。

注意：

  1. Java中变量必须先声明后使用。
  2. 变量只有在其作用域（其定义所在的一对{}内）内才有效。
  3. 同一个作用域内，不能定义重名变量。

Java语言支持的变量类型有：

  • 类变量：独立于方法之外的变量，用 static 修饰。
  • 实例变量：独立于方法之外的变量，不过没有 static 修饰。
  • 局部变量：类的方法中的变量。

数据类型

基本数据类型

整形
类型 存储大小 取值范围
byte 1字节 -128~127
short 2字节 -2¹⁵~2¹⁵-1
int 4字节 -2³¹~2³¹-1
long 8字节 -2⁶³~2⁶³-1

注：1. java中整形变量默认为int，声明long型后要加l或L；

浮点型

（带小数点的数据）

类型 存储大小 取值范围
float 4字节
double 8字节

注：

  1. float尾数可以精确到7位有效数字.
  2. java浮点型常量默认为double，声明float须后加f或F
  3. float存储大小比long小，但取值范围比long大
字符型

（表示通常意义上的字符）

类型 存储大小 取值范围
char 1字节

注：

  1. 定义char型‘ ’内只能有一个字符。
  2. java中允许转义字符‘\’将其后的字符转变为字符型常量。 如char c=‘\n’
  3. 可以直接使用Unicode值表示字符型常量 。 如char c=’\u0043’ (C的Unicode码)
布尔型

Boolean 只能取 false或true，常在条件判断或循环结构中使用。

基本数据类型之间的运算规则

前提：boolean型不参加运算。

  1. 自动类型提升：当容量小的数据类型的变量与容量大的数据类型的变量做运算时，结果自动提升为容量大的数据类型。

    byte 、char、short→ int → long → float→ double

    注：当byte 、char、short三种变量做运算时，结果为int类型。

  1. 强制类型转换：自动类型提升的逆运算。

    1.使用强转符：（）

    2.强制类型转换可能导致精度损失。

  2. ①整形常量默认为int型，浮点型常量默认为double型。

字符串类型（String）

1.声明String变量时，使用一对“”

2.String可以和8种基本数据类型变量做运算，只能是用+做连接运算。

3.运算的结果仍然是String类型，+号前后只要出现string类型，这个+号及运算顺序在他后面的+号就都表示连接。

4.常用方法：

substring：从一个较大的字符串中提取一个字符

int length（）：返回字符串的长度
