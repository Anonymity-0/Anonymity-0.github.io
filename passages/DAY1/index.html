DAY1

常用dos命令

dir ：列出当前目录下的文件以及文件夹

md：创建目录

rd：删除目录 只能删除空的

cd：进入指定目录

cd ..：退回到上一级目录

cd\：返回到根目录

del：删除文件 tip：del+*.格式可以将目录下同一格式所有文件删除

exit：退出dos命令行

输出简单程序hello world

1.源文件以.java结尾。

2.源文件中可以有多个class声明的类。

3.类中可以有主方法（main（）方法），其格式是固定的：public static void main(String[ ] args){}

4.main()方法是程序的入口，方法内是程序的执行部分（有点像c语言的函数）

5.一个源文件中只能有一个声明为public 的类，同时要求此类的类名与源文件名一致。

6.每个语句都已；结束。

7.执行程序：编译：javac.exe 编译完，生成诸多个.class字节码文件。 运行java。exe。

注释

单行注释

格式：

//注释文字

多行注释

格式：

/*

注释文字

*/

示例：

/*

这是

好多行

好多行

好多行

……

注释

*/

注：

1.对于单行和多行注释，被注释的文字不会被JVM解释执行。

2.多行注释里不允许有多行注释嵌套，如：

/*

/*

不许嵌套

*/

*/

文档注释（java特有）

格式：

/**

*@author 指定java程序的作者

*@version 指定源文件的版本

*@param 方法的参数信息说明

*/

示例：

/**这是我的第一个java程序

@author qjj

@version v1.0

*/

文档注释内容可以被javadoc所解析，生成一套以网页文件形式体现的该程序的说明文档。

多注释，多注释，多注释，不然过两天自己都看不懂
