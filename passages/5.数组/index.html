5.数组

  1. 数组属于引用数据类型变量，数组的元素既可以是引用数据类型，又可以是基本数据类型
  2. 创建数组对象会在内存中开辟一整块连续的空间
  3. 数组的长度一旦确定，就不能修改。

一维数组

1.声明和初始化:

声明：数组元素类型 [] 数组名

①静态初始化(数组的初始化和数组元素的赋值操作同时进行)

如: int[]ids;

​ ids= new int[]{1,2,3,4};

②动态初始化（数组的初始化和数组元素的赋值分开进行）

如：String[] names = new String[5];

注：java的数组长度不要求是常量，new int[n]会创建一个长度为n的数组。

2.如何获取数组的长度

属性：length

例如：ids.length。

3.遍历数组

for(int i = 0 ; i< ids.length;i++){

}

4.数组元素默认初始值

①数组元素是整型： 0；

②数组元素是浮点型：0.0

③数组元素是char型：0或’\u0000’，而非‘0‘

④数组元素是boolean型：false

⑤数组元素是引用数据类型：null
